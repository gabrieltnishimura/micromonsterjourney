------/------/------/------ Right Click on flash ------/------/------/------
Programa em AS3 que desativa a janelinha que abre quando
se clica com botao direito em um programa em flash.

Ele ainda passa a requisicao de botao direito para o flash container
para que o flash possa usar como mais um input.

P.S.:
Due to the use of Flash Player's External Interface the included code 
runs only in a web server. Please place the contents of bin folder to the 
web server's directory and access it via http://.
